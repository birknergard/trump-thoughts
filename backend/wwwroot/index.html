Trump API

Welcome to the TrumpAPI! This page contains documentation for the various API endpoints related to Donald Trump.

Briefly

On every endpoint except ImageUpload, the id is a unique (to its API) integer.

Personell Endpoint

Endpoint: /api/Personell

Data Structure for Entry (JSON):

{
        "id": 0,
        "name": "Donald Trump",
        "age": 77,
        "title": "President",
        "description": "45th President of the United States."
    }

Methods

POST

Creates a new entry for personnel. Requires this JSON structure:

{
        "id": 0,
        "name": "Donald Trump",
        "age": 77,
        "title": "President",
        "description": "45th President of the United States."
    }

GET

Fetches all entries from the API. Example:

[
        { "id": 1, "name": "Donald Trump", "age": 77, "title": "President", "description": "45th President of the United States." },
        { "id": 2, "name": "Ivanka Trump", "age": 42, "title": "Businesswoman", "description": "Daughter of Donald Trump" }
    ]

GET by ID

Endpoint: /api/Personell/{id}

Fetches an entry by its unique id. Response structure:

{
        "id": 1,
        "name": "Donald Trump",
        "age": 77,
        "title": "President",
        "description": "45th President of the United States."
    }

PUT

Endpoint: /api/Personell/{id}

Modifies an existing entry. Uses id as an identifier.

DELETE

Endpoint: /api/Personell/{id}

Deletes an entry by its unique id.

Merch Endpoint

Endpoint: /api/Merch

Data Structure for Entry (JSON):

{
        "id": 0,
        "price": 20.00,
        "itemName": "Make America Great Again Hat",
        "description": "A red cap with a powerful message."
    }

Methods

POST

Creates a new inventory item. Requires this JSON structure:

{
        "id": 0,
        "price": 20.00,
        "itemName": "Make America Great Again Hat",
        "description": "A red cap with a powerful message."
    }

GET

Fetches all inventory items from the API. Example:

[
        { "id": 0, "price": 20.00, "itemName": "Make America Great Again Hat", "description": "A red cap with a powerful message." },
        { "id": 1, "price": 25.00, "itemName": "Trump T-shirt", "description": "A shirt to show your support." }
    ]

GET by ID

Endpoint: /api/Merch/{id}

Fetches an inventory item by its unique id. Response structure:

{
        "id": 0,
        "price": 20.00,
        "itemName": "Make America Great Again Hat",
        "description": "A red cap with a powerful message."
    }

PUT

Endpoint: /api/Merch/{id}

Modifies an existing inventory item. Uses id as an identifier.

DELETE

Endpoint: /api/Merch/{id}

Deletes an inventory item by its unique id.

Thought Endpoint

Endpoint: /api/Thought

Data Structure for Entry (JSON):

{
        "id": 0,
        "imageUrl": "string",
        "statement": "Make America Great Again!",
        "topic": "Politics",
        "title": "Trump's Statement",
        "tone": "Inspirational"
    }

Methods

GET

Fetches all thoughts from the API. Example:

[
        {
            "id": 1,
            "imageUrl": "https://example.com/image1.jpg",
            "statement": "Make America Great Again!",
            "topic": "Politics",
            "title": "Trump's Statement",
            "tone": "Inspirational"
        }
    ]

GET by ID

Endpoint: /api/Thought/id={id}

Fetches a specific thought entry by its unique id. Response structure:

{
        "id": 1,
        "imageUrl": "https://example.com/image1.jpg",
        "statement": "Make America Great Again!",
        "topic": "Politics",
        "title": "Trump's Statement",
        "tone": "Inspirational"
    }

GET by Topic

Endpoint: /api/Thought/topic={topic}

Fetches thoughts filtered by a specific topic. Example response:

[
        {
            "id": 2,
            "imageUrl": "https://example.com/image2.jpg",
            "statement": "Winning is everything.",
            "topic": "Motivation",
            "title": "Trump's Winning Mindset",
            "tone": "Inspirational"
        }
    ]

GET by Tone and Topic

Endpoint: /api/Thought/multi/{tone}+{topic}

Fetches thoughts filtered by both tone and topic. Example response:

[
        {
            "id": 3,
            "imageUrl": "https://example.com/image3.jpg",
            "statement": "You're going to love it!",
            "topic": "Motivation",
            "title": "Trump's Promise",
            "tone": "Positive"
        }
    ]

PUT

Endpoint: /api/Thought/{id}

Modifies an existing thought entry. Uses id as an identifier.

DELETE

Endpoint: /api/Thought/{id}

Deletes a specific thought entry by its unique id.

ImageUpload API

Endpoint: /api/UploadImage

Methods

POST: Direct

Endpoint: /api/UploadImage

Uploads an image to the API.

POST: Temporary image

Endpoint: /api/UploadImage/temp

Uploads an image to the temp directory. Works similarly to POST: /UploadImage.

POST: Process temporary image

Endpoint: /api/UploadImage/process/{imageUrl}

Moves an image from the temp folder to the images folder. Uses imageUrl as the identifier.

DELETE from images

Endpoint: /api/UploadImage/{imageUrl}

Deletes an image directly from the images folder.

DELETE from temp

Endpoint: /api/UploadImage/{imageUrl}

Deletes an image from the temp folder.
